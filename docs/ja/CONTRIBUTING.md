# Contributing

This is a framework and template resource. Contributions are welcome!

## How to contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/xyz`)
3. Commit your changes
4. Push to the branch
5. Open a Pull Request

## Guidelines

- Keep templates and checklists concise
- Test new domain resources before submitting
- Include links to authoritative sources
- Follow existing format conventions

Thanks!
